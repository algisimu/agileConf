### Install


```
$ npm install
$ npm install -g tsd@^0.6.0
$ npm install -g typescript@~1.6.2
$ npm install -g bower
$ npm install -g grunt-cli
$ bower install
$ grunt install
```

### Compile


```
$ grunt compile
```


### Run


```
$ grunt serve
```

### Update after code changes

```
$ npm install
$ bower install
$ grunt install
```
