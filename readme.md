Install

======


```
$ npm install -g tsd@^0.6.0
$ npm install -g typescript@~1.6.2
$ npm install -g bower
$ npm install -g grunt-cli
$ grunt exec:install_definitions
$ bower install
```

Compile

======


```
$ grunt compile
```


Run

======


```
$ grunt serve
```
